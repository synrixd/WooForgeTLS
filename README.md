# 🚀 WooForgeTLS

> Generador profesional de productos digitales para WooCommerce  
> Optimizado para SEO (RankMath) + flujo rápido de carga masiva

---

## ✨ Características

- 🧩 Interfaz basada en **cards (tarjetas)**
- 📦 Exportación directa a **CSV compatible con WooCommerce**
- 📄 Importar / exportar productos en **JSON**
- ⚡ Flujo rápido para crear múltiples productos
- 🧠 Validación básica de campos
- 🖥️ Versión **desktop (.exe)**

---

## 🛠️ Stack

- Python (Flask)
- JavaScript (Vanilla)
- HTML + CSS

---

## ⚙️ Instalación

```bash
git clone https://github.com/synrixd/WooForgeTLS.git
cd WooForgeTLS
pip install -r requirements.txt
python app.py
```

---

## 🧠 Cómo usar

### 🧩 Flujo básico

1. Crear productos usando las tarjetas (cards)
2. Completar los campos necesarios
3. Exportar a CSV
4. Importar en WooCommerce

---

## 🧾 Sobre los campos

### ⚠️ IMPORTANTE
Los campos que ya vienen con valores por defecto:

👉 **NO deben modificarse**, ya que están optimizados para:
- compatibilidad con WooCommerce
- estructura correcta del CSV

---

## 🧠 SEO (RankMath)

Este generador está pensado para trabajar con:

👉 **RankMath**

Pero ten en cuenta:

- El score (ej: 98/100) **NO depende de la herramienta**
- Depende de cómo escribas:
  - título
  - descripción
  - keywords

👉 WooForgeTLS solo estructura el contenido correctamente en HTML.

---

## 🖼️ Imágenes

Las imágenes deben:

- Estar subidas previamente en tu hosting / CDN
- Ser URLs accesibles públicamente

### 📌 Formato correcto:

img1.png, img2.png, img3.png

👉 Separadas por comas  

---

## 📥 Archivos descargables

Este sistema está diseñado **solo para productos digitales**.

### ✔️ Ejemplos:
- PDFs
- ZIPs
- Software
- Recursos descargables

### 📌 Importante:

Debes colocar:

- enlace directo de descarga
- archivo previamente subido a tu servidor o nube

---

## ❌ No está pensado para:

- productos físicos
- envíos
- inventario físico
